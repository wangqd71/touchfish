# DarkTaskRPG - 暗黑任务栏RPG

一款运行在 Windows 任务栏中的暗黑风格放置类 RPG 游戏。灵感来源于 Steam 上的 [TBH: Task Bar Hero](https://store.steampowered.com/app/3678970/)。

## 游戏特色

- **任务栏运行**：窗口置顶无边框，可拖动，不影响正常工作
- **6个职业**：战士、法师、游侠、骑士、牧师、刺客
- **90个技能**：每职业3条分支，每分支5个技能
- **7种装备稀有度**：普通 → 优秀 → 稀有 → 史诗 → 传说 → 神话 → 宇宙
- **暗黑词条系统**：28种通用词条 + 90种技能加成词条
- **3难度 × 3章节**：普通 → 噩梦 → 地狱，共90关
- **自动战斗**：挂机自动推进，适合碎片时间
- **存档系统**：自动存档，支持读档

## 安装运行

```bash
# 安装依赖
pip install PyQt5

# 启动游戏
python main.py
```

或双击 `启动游戏.bat`。

## 游戏系统

### 关卡结构

```
普通 (S1-33)           噩梦 (S34-66)           地狱 (S67-100)
┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│ Ch1 黑暗森林 │       │ Ch1 黑暗森林 │       │ Ch1 黑暗森林 │
│ Ch2 废弃矿洞 │       │ Ch2 废弃矿洞 │       │ Ch2 废弃矿洞 │
│ Ch3 恶魔城堡 │       │ Ch3 恶魔城堡 │       │ Ch3 恶魔城堡 │
└─────────────┘       └─────────────┘       └─────────────┘
 30只怪+1Boss           怪×1.3倍率              怪×1.7倍率
```

### 经验曲线

混合公式：`EXP_BASE × 1.08^(Lv-1) + POLY × (Lv-10)^2`（Lv10后生效）

| 等级 | 所需经验 | 累计时间 |
|------|----------|----------|
| Lv10 | 599 | ~10分钟 |
| Lv30 | 82,795 | ~15小时 |
| Lv50 | 333,028 | ~69小时 |
| Lv100 | 2,231,044 | ~435小时 |

### 装备稀有度

| 稀有度 | 词条数 | 技能词条概率 |
|--------|--------|-------------|
| 普通 | 0 | 0% |
| 优秀 | 1 | 0% |
| 稀有 | 2 | 15% |
| 史诗 | 3 | 25% |
| 传说 | 4 | 35% |
| 神话 | 5 | 45% |
| 宇宙 | 6 | 55% |
